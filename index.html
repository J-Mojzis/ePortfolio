Work Samples

Explore a collection of data-driven accomplishments, that demonstrate the essence of my work

Student Registration Plattform

This project aims to create a secure web-based platform for student registrations that reduces media breaks, manual data entry errors, and process time, while also providing an easy-to-use interface for both users and administrators. (Group Project)

Numerical Approximation with Gradient Descent

The objective of this project is to cultivate a fundamental comprehension of numerical approximation techniques in higher dimensions, with a specific emphasis on Gradient Descent and its practical implementation. The project involves utilizing the MNIST-dataset to recognize handwritten numbers. (Group Project)

Fourier Transformations: Analyzing Sun Data, Music, and Images

The objective of this task is to develop a fundamental understanding of Fourier transformation for approximating periodic functions and its practical application in data analysis and processing. Sequentially, we will analyze and process sun data, music, and images using Fourier transformations. (Group Project)
